## 實驗介紹
### 操作手法跟目的
- 黑腹果蠅 (*Drosophila melanogaster*)，屬於遺傳學研究中廣泛使用的model organism，其優勢包括: 
  - 世代的生命週期短
  - 子代數量多
  - 染色體數目簡單（2n=8）
  - 具備多種易於辨識之突變性狀
- 本實驗利用果蠅之性聯遺傳與體染色體顯性遺傳特性，探討不同突變品系雜交後子代性狀之分離比例，並透過棋盤方格法（Punnett square）推論子代理論比例，並以卡方檢定（chi-square test）比較觀察值與理論值之間是否具顯著差異，驗證孟德爾遺傳法則之適用性。
- - 孟德爾遺傳法則包含以下原則：
  - 分離律：每一對等位基因在形成配子時會彼此分離。
  - 獨立分配律：位於不同對染色體上的基因，其遺傳過程互不干擾。
- 處女蠅的選取：
  - 由於雌蠅羽化後6小時內不具交配能力且不帶精子，因此可以透過收集6小時內內羽化的成蟲，確保收集的果蠅是 "處女蠅"，以確保能夠收集到期望的雜交結果

## 棋盤方格法 (Punnett Square)
### CyO/捲翅
- 白眼基因為**X 染色體性聯隱性遺傳**（Xʷ = 白眼，X⁺ = 紅眼） 
- 捲翅基因為**第二對染色體顯性遺傳**（Cy = 捲翅，但 Cy/Cy 致死，故母蠅只能為 Cy/+） 
- 正常翅為隱性 (+/+)

#### F1 Punnett Square

| | **母配子 (1/2)**<br>**Xʷ ; Cy** | **母配子 (1/2)**<br>**Xʷ ; +** |
|:-:|:-:|:-:|
| **公配子 (1/2)**<br>**X⁺ ; +** | **Xʷ/X⁺ ; Cy/+**<br>♀ 紅眼捲翅 | **Xʷ/X⁺ ; +/+**<br>♀ 紅眼正常|
| **公配子 (1/2)**<br>**Y ; +** | **Xʷ/Y ; Cy/+**<br>♂ 白眼捲翅 | **Xʷ/Y ; +/+**<br>♂ 白眼正常 |


#### F2 Punnett Square（4×4 = 16 格）

- 母配子列在左邊，公配子列在上面：

| 母配子 ↓ \ 公配子 → | Xʷ ; Cy (1/4) | Xʷ ; + (1/4) | Y ; Cy (1/4) | Y ; + (1/4) |
|----------------------|---------------|--------------|--------------|-------------|
| **Xʷ ; Cy (1/4)** | **Xʷ/Xʷ ; Cy/Cy** <br> **♀ 白眼 死亡** | **Xʷ/Xʷ ; Cy/+** <br> ♀ 白眼捲翅 | **Xʷ/Y ; Cy/Cy** <br> **♂ 白眼 死亡** | **Xʷ/Y ; Cy/+** <br> ♂ 白眼捲翅 |
| **Xʷ ; + (1/4)** | **Xʷ/Xʷ ; Cy/+** <br> ♀ 白眼捲翅 | **Xʷ/Xʷ ; +/+** <br> ♀ 白眼正常 | **Xʷ/Y ; Cy/+** <br> ♂ 白眼捲翅 | **Xʷ/Y ; +/+** <br> ♂ 白眼正常 |
| **X⁺ ; Cy (1/4)** | **X⁺/Xʷ ; Cy/Cy** <br> **♀ 紅眼 死亡** | **X⁺/Xʷ ; Cy/+** <br> ♀ 紅眼捲翅 | **X⁺/Y ; Cy/Cy** <br> **♂ 紅眼 死亡** | **X⁺/Y ; Cy/+** <br> ♂ 紅眼捲翅 |
| **X⁺ ; + (1/4)** | **X⁺/Xʷ ; Cy/+** <br> ♀ 紅眼捲翅 | **X⁺/Xʷ ; +/+** <br> ♀ 紅眼正常 | **X⁺/Y ; Cy/+** <br> ♂ 紅眼捲翅 | **X⁺/Y ; +/+** <br> ♂ 紅眼正常 |

> [!Caution]
上面標「死亡」的格子（Cy/Cy）會在幼蟲或蛹期死掉，**不會有成蟲**。所以 F2 成蟲只來自 **16 格中的 12 格** (4 格死亡)

## 結果說明
### Cyo/捲翅
#### R跟卡方檢定
- 我們最後收集到的F1果蠅數量如下: 

|品系	|公，白眼，捲翅| 公，白眼，正常翅|母，紅眼，捲翅|母，紅眼，正常翅|
|---|---|---|---|---|
|**個體數量**| 32|15|6	|13|

- 利用R來做檢定分析: 

```r
# 果蠅數據
observed <- c(32, 15, 6, 13)   # 公捲, 公正, 母捲, 母正
expected <- rep(sum(observed)/4, 4)  

# 算卡方
chisq_result <- chisq.test(observed, p = rep(0.25, 4))

# 看結果
print(chisq_result)

# 殘差檢查
residuals <- chisq_result$residuals
names(residuals) <- c("♂白眼捲翅", "♂白眼正常翅", "♀紅眼捲翅", "♀紅眼正常翅")
print(residuals)

# 長條圖
barplot(observed, 
        names.arg = c("♂捲翅", "♂正常", "♀捲翅", "♀正常"),
        col = c("steelblue", "lightblue", "coral", "lightcoral"),
        main = "果蠅F1性狀分布",
        ylab = "number",
        ylim = c(0, 35))
abline(h = 16.5, lty = 2, col = "red", lwd = 2)
legend("topright", "理論值 16.5", lty = 2, col = "red", bty = "n")
```

- 計算結果如下: 

| 性狀組合 | 觀察值 (O) | 理論值 (E) | (O-E) | (O-E)² | (O-E)²/E |
|----------|-----------|-----------|-------|--------|-----------|
| ♂ 白眼捲翅 | 32 | 16.5 | 15.5 | 240.25 | **14.56** |
| ♂ 白眼正常翅 | 15 | 16.5 | -1.5 | 2.25 | **0.14** |
| ♀ 紅眼捲翅 | 6 | 16.5 | -10.5 | 110.25 | **6.68** |
| ♀ 紅眼正常翅 | 13 | 16.5 | -3.5 | 12.25 | **0.74** |
| **總計** | **66** | **66** | 0 | — | **χ² = 22.12** |

- 得到以下卡方值: 

$$\chi^2 = \sum \frac{(O - E)^2}{E} = 14.56 + 0.14 + 6.68 + 0.74 = 22.12$$

- 其中: 
  - **自由度 (df)** = 組數 − 1 = 4 − 1 = **3**
  - **χ² = 22.12**
  - **p < 0.001**（對照卡方分布表，χ² = 22.12 遠大於 16.27，對應 p < 0.001）

$$df = 3,\ \text{p-value} = 6.155e-05$$

- 卡方檢定結果顯示，χ²(3) = 22.12，**p < 0.001**，達統計顯著水準。
- 表示觀察到的性狀分佈比例**顯著偏離**孟德爾遺傳定律預期之 1:1:1:1 分離比。

## 問題探討
### yw無法在期限內產生F1的可能原因
#### 培養基脫落
- 通常在採收果蠅時有些果蠅不一定會因為負趨地性而往上爬（可能剛羽化），所以常常在收集時，透過拍擊試管讓其掉下來，以加速進程，但是此時的培養基可能因此而脫落、覆蓋瓶壁，對採集造成影響。
- 培養基的脫落除了影響很多尚未化蛹的蛆以外，還會使本來的蛹因此而被培養基給罩住，導致窒息而亡，進而顯著降低羽化率
- 因此，於4/29下午將該批種原瓶移入待滅區，提前終止收集。此操作亦可能導致本組可採收之F1子代總數低於預期

#### 隨機運氣導致
- 後期剩下的一管yw1118種原瓶，在中午的時候往往可以發現大量的，前一夜羽化的果蠅，但是在種原瓶釋放，五個小時過後卻不一定能夠等代到羽化的新成蟲
- 也就是說果蠅產出的時間往往可能跟實驗採收的時間錯開，於更晚的時候才羽化

#### 收集過程失誤造成的逃逸損失
- 果蠅在收集或麻醉期間可能因為動作不準確或麻醉時間過短，造成部分個體可能飛離或在轉移過程中遺失。此些失誤可能造成yw的親代收集進度緩慢，對後續進度造成影響

#### 親代繁殖效率不足
- 雖然成功將 yw1118 處女蠅與 OR 公蠅置於同一培養瓶中，但不代表一定能有效完成交配與產卵。若 OR 公蠅因麻醉造成活性下降、年齡較大或尚未完全性成熟，可能降低交配成功率
- 除此之外，因培養瓶內親代數量偏少，則整體交配次數及產卵量也可能下降。上述因素皆可能使此次實驗的yw較晚交配，受精卵數量也可能不足，導致最終 F1 子代數低於預期

#### 蛹附著位置影響羽化成功率
- 果蠅幼蟲於進入蛹期前，通常會離開培養基表面並移動至瓶壁、棉塞附近或較乾燥的位置化蛹。然而，不同個體的附著位置可能存在差異，若部分蛹附著於培養基邊緣、過於潮濕的區域，或因培養基脫落而遭覆蓋，可能影響其正常羽化過程。
- 尤其部分結蛹的位置較靠近試管上側、接近封口棉花的位置，在實驗過程中因需開啟與關閉棉塞進行釋放、收集及觀察操作，棉花可能與蛹直接接觸，造成擠壓、摩擦或拉扯。若蛹附著於棉塞纖維上，則於取出棉塞時可能被一併帶出，甚至造成蛹脫落或受損。
- 以上都可能影響後續正常發育與羽化成功率，使部分個體無法順利羽化，導致實際收集數量低下，進一步造成無法在有限時間內收集足夠F1子代的後果。

#### OR（野生型公果蠅）對乙醚的劑量承受度
- 本組在4/29日時才採收到第一隻yw1118處女蠅。以當時情況來看，根據果蠅10~11天才會成蟲來說，有機會在實驗完成日5/11前，成功的收集到yw1118和OR的F1子代，但由於OR的提早死亡，導致最終並未能在期限內收集足夠的F1後代。yw1118裡所配置的OR，常常尚未跟雌蠅交配前就死亡，推測是OR受到過多的乙醚導致
- 在這次實驗裡，A班跟B班的所有小組共享OR，很有可能拿到的OR時，其他組先前用了乙醚麻醉過，等到採集夠了，便把剩下的OR給其他組別的學生使用，或是重新放回種原瓶。這可能導致下一次的組別在領取OR時，會重複麻醉同一隻OR雄蠅
- 另一個原因也可能是源自於，在顯微鏡下觀察果蠅的性別跟性狀時，果蠅會因為乙醚藥效退去而醒來，所以需要重覆用乙醚麻醉，在麻醉時使用過多的劑量，可能對OR造成不可逆的傷害

#### 羽化時間不同步
- 本次實驗在都是中午釋放，五小時後收集，並且大概收集了三天 (05/08~05/11)，然而，不同性狀之果蠅羽化時間存在差異，固定時間收集造成取樣偏差，尤其雌性捲翅個體可能較晚羽化
- 推測兩個性別的果蠅羽化的時間有所差異，例如: 
  - 第一批跟蒐集的: ♂白眼捲翅為主
  - 第二批蒐集的: ♂白眼正常翅 + ♀紅眼正常翅
  - 最後一批蒐集的: ♀紅眼捲翅 (但是蒐集數目少，其他的可能還在蛹裡)
- 如欲解決羽化時間不同步的問題，除了可以嘗試連續收集，每天固定時間收，也可以增加收集F1子代的時長，增加可能收集到的母果蠅機率

- 剛羽化的親代及F1果蠅顏色通常呈現黃色，較淺的體色可能導致觀察性梳 (sex comb，雄性成蠅前腳的一排黑色的鬃毛) 時出現困難，進而誤判果蠅性別

## 結論
- 本次果蠅遺傳實驗原先以 FM7a 與 Dr 品系作為實驗材料進行遺傳分析，但因前期親代收集狀況不理想，因此後續改以 yw1118 與 CyO 品系進行交配及子代性狀分析，以觀察果蠅遺傳特性及其與孟德爾遺傳比例之間的關係。藉由觀察子代之眼色、翅型及性別等性狀，了解基因遺傳模式及平衡染色體在果蠅遺傳實驗中的應用。

- 實驗過程中，本組未能於有限時間內成功收集足夠之 yw × OR  的F1子代，因此未能完成預期的後續分析。經討論後推測，其主要原因可能來自培養基脫落造成幼蟲及蛹存活率下降、收集過程中個體損失、親代繁殖效率不足、蛹附著位置影響羽化成功率，以及 OR 公蠅可能因重複乙醚麻醉而降低存活率等因素，進而導致親代數量不足及子代產生量下降。此外，由於果蠅發育時間具有一定個體差異，加上實驗時程限制，也可能使部分個體尚未完成羽化便結束實驗，最終造成可收集的 F1 子代數量低於預期。

- 此外，在 CyO 檢定結果中，觀察數據與理論比例之間出現明顯差異，其中以 ♂白眼捲翅個體數量偏高，以及 ♀紅眼捲翅個體數量偏低最為顯著。推測造成此現象的原因包括不同性狀個體羽化時間不同步、固定時間收集所造成的取樣偏差，以及性狀與性別辨識過程中的人為誤差等因素。此外，果蠅本身生長發育具有生物個體差異，而樣本數有限也可能增加隨機誤差對結果的影響，因此實際數據未必能完全符合理論預測值。

- 本次實驗中，我們了解到果蠅遺傳結果除了受到基因型本身控制外，實驗操作流程、培養條件、收集方式以及結果判讀方法等因素，皆可能對最終實驗結果產生明顯影響。同時也發現，理論上的遺傳比例在實際實驗中常受到生物個體差異與操作誤差干擾，因此需透過適當控制實驗條件以提高數據可信度。

- 若未來進行相同實驗並提高實驗成功率及數據可靠性，可於實驗流程中減少乙醚重複麻醉次數，並控制麻醉劑量及暴露時間，以降低對果蠅存活率及活動能力之影響。於子代收集階段，可增加每日收集次數並適當延長收集天數，減少不同基因型及性別因羽化時間差異所造成之偏差。此外，在操作過程中應避免過度拍擊試管，以降低培養基脫落及蛹受損機率，並於開啟棉塞時減少對附著蛹的擠壓與拉扯。在性狀判讀方面，可增加觀察時間並透過組員交叉確認，以降低誤判率，進一步提升統計結果及數據分析的準確性與可靠性。
